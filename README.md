# 🏗️ מגדל האיזון | Tower Balance Game

משחק איזון פיזיקלי מתקדם מבוסס PWA עם מנוע פיזיקה מלא וחישובי מרכז מסה בזמן אמת.

## 🎮 תיאור המשחק

משחק מבוסס מיומנות בו השחקן מניח דמויות (תיירים) על מרפסות מגדל רב-קומתי תוך ניהול מרכז המסה, מומנט והתפלגות משקל בזמן אמת. המטרה: להגיע לניקוד הגבוה ביותר מבלי שהמגדל יקרוס!

## ✨ תכונות עיקריות

### מנוע פיזיקה מתקדם
- **חישוב מרכז מסה דינמי** - חישוב COM משולב בזמן אמת
- **תאוצה זוויתית** - מומנט התמד ותנועה זוויתית ריאליסטית
- **מכניקת החלקה** - דמויות מחליקות בזוויות קריטיות
- **אימפקט נפילה** - השפעת גובה נפילה על יציבות המגדל
- **כוח רוח** - גורם סביבתי משתנה (ברמות קושי גבוהות)

### מערכת דמויות
- **תייר רגיל** 🧍 - משקל: 1, בסיס המשחק
- **תייר כבד** 🏋️ - משקל: 2, מופיע כל 5 תורות
- **ילד** 👶 - משקל: 0.5, מופיע אקראית

### רמות קושי
1. **קל** - ללא טיימר, ללא רוח
2. **בינוני** - טיימר 15 שניות, ללא רוח
3. **קשה** - טיימר 10 שניות + רוח
4. **מומחה** - טיימר 8 שניות + רוח תדירה + חיכוך נמוך

### מצבי מצלמה
- **מבט רגיל** - פרספקטיבה תלת-ממדית
- **מבט על** ⬇️ - תצוגה מלמעלה עם סימון COM
- **מבט אופק** ↔️ - דגש על זווית הטיה
- **איפוס מצלמה** 🎯 - חזרה למצב ברירת מחדל

### UI/UX מתקדם
- **Ghost Preview** - תצוגה מקדימה של מיקום התייר
- **מד יציבות** - ויזואליזציה של מצב המגדל
- **טיימר ויזואלי** - ספירה לאחור דינמית
- **אנימציות חלקות** - חוויית משחק מהנה
- **תמיכה במחוות** - Pinch, Drag, Tap

## 🚀 התקנה והרצה

### דרישות מערכת
- דפדפן מודרני עם תמיכה ב-HTML5 Canvas
- תמיכה ב-Service Worker (לפונקציונליות PWA)
- מומלץ: מכשיר עם מסך מגע

### הרצה מקומית

#### שיטה 1: Python HTTP Server
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

לאחר מכן פתח בדפדפן:
```
http://localhost:8000/balance-tower-game.html
```

#### שיטה 2: Node.js HTTP Server
```bash
npx http-server -p 8000
```

#### שיטה 3: VS Code Live Server
1. התקן את תוסף "Live Server"
2. לחץ לחיצה ימנית על `balance-tower-game.html`
3. בחר "Open with Live Server"

### התקנה כ-PWA

#### במובייל (Android/iOS)
1. פתח את המשחק בדפדפן
2. ב-Chrome: לחץ על תפריט (⋮) → "Add to Home screen"
3. ב-Safari (iOS): לחץ על Share → "Add to Home Screen"

#### במחשב
1. פתח את המשחק ב-Chrome/Edge
2. לחץ על אייקון ההתקנה בשורת הכתובת
3. לחץ "Install"

## 🔧 התאמה אישית

### שינוי פרמטרים פיזיקליים
ערוך את אובייקט `CONFIG` בקובץ ה-HTML:

```javascript
const CONFIG = {
    physics: {
        gravity: 9.81,        // תאוצת כובד
        maxAngle: 45,         // זווית מקסימלית לפני קריסה
        damping: 0.95,        // דעיכת תנועה
        friction: 0.7,        // חיכוך
    },
    tower: {
        floorCount: 8,        // מספר קומות
        maxPerFloor: 4,       // מקסימום דמויות לקומה
    },
    // ...
};
```

characters: {
